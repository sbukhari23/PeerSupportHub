# PeerSupportHub – Peer Support Platform

## Purpose
A peer support platform designed to help students and early professionals build habits, stay focused, and find direction through community support, habit tracking, and mentorship.

---

## Architecture
- Full-stack application with separated frontend and backend  
- **Backend:** Node.js / Express API with MongoDB  
- **Frontend:** React Single Page App (SPA) with Vite build tool  
- **Real-time features:** Socket.IO for messaging and notifications  

---

## Tech Stack

### Backend
- Node.js, Express.js, MongoDB (Mongoose)
- Socket.IO, JWT authentication, bcrypt
- Helmet (security), CORS, rate limiting
- File upload handling (avatars)

### Frontend
- React 19, React Router DOM
- Vite (build tool), TailwindCSS (styling)
- Radix UI components, Lucide React icons
- Axios (API calls), React Toastify (notifications)

---

## Core Features
- **Habit Tracking:** Create, track, and log daily habits with streaks
- **Peer Groups:** Join groups for accountability and support
- **Mentorship:** Connect with mentors for 1-on-1 guidance
- **Real-time Messaging:** Chat with buddies and group members
- **Community Challenges:** Participate in group challenges
- **Reflection Tools:** Personal reflection entries
- **Admin Panel:** User management and content moderation
- **Membership Tiers:** Free, Growth, and Premium plans

---

## UI / UX
- Clean, modern design using TailwindCSS
- Responsive layout for mobile and desktop
- Dashboard-centric interface with navigation sidebar
- Card-based component design
- Toast notifications for user feedback

Students and young professionals struggling with procrastination, excessive screen time, and consistency issues, supported through community accountability and structured habit-building tools.
